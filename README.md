# Image Background Remover

A web application to remove backgrounds from images using AI.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- @imgly/background-removal

## Getting Started

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- Upload images
- AI-powered background removal
- Download processed images
- Responsive design
